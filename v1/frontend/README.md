# imtdb
IMTDb frontend application
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
Is served at localhost:8080
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
